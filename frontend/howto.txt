lauch bootstrap doc locally:
Run npm install to install the Node.js dependencies, including Hugo (the site builder).
Run npm run test (or a specific npm script) to rebuild distributed CSS and JavaScript files, as well as our docs assets.
From the root /bootstrap directory, run npm run docs-serve in the command line.
Open http://localhost:9001/ in your browser, and voilà.



lauch deamonite material doc locally:
Run through the tooling setup to install Jekyll and other Ruby dependencies with bundle install.
Run npm install to install Node.js dependencies.
Run npm run dist and npm run docs to rebuild distributed CSS and JavaScript files, as well as docs assets.
From the root /daemonite-material directory, run npm run docs-serve in the command line.
Open http://localhost:9999/material in your browser, and voilà.
